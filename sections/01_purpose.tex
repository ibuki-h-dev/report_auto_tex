\section{実験目的}
本実験の目的は、コンピュータを用いた自動計測システムの基礎を修得することである。具体的には、以下の項目を目的とする。

\begin{itemize}
    \item システム開発ソフトウェアであるLabVIEWを用い、ファンクションジェネレータおよびデジタルマルチメータを統合的に制御するプログラムを作成することで、自動計測の原理と手法を理解する。
    \item 通信インターフェースを介した計測器とPC間のデータ転送について、その物理的な接続および設定方法を学ぶ。
    \item 半導体素子であるダイオードの電圧—電流静特性を実測し、得られたデータを解析することで素子の基本的な電気的特性を理解する。
    \item 手動計測による結果と比較し、自動計測の有用性について考察する。
\end{itemize}
